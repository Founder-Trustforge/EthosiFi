## What is EthosiFi?

EthosiFi is an **on-chain reputation protocol** that awards **Genesis TrustScores** (soulbound NFTs) to builders who demonstrate verifiable ethical behavior ‚Äî no PDFs, no KYC, only on-chain proof + optional ZK privacy.

The system aims to become a **trust & safety layer** for DeFi, DAOs, launchpads, grants, and community treasuries.
